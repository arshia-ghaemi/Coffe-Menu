# ☕ Coffee Menu App

A modern coffee shop menu built with React and real API integration.  
This project demonstrates working with APIs, dynamic data rendering, and building a responsive UI.

---

## ✨ Features

- 📡 Fetch coffee products from real API
- 🧾 Dynamic menu rendering
- ⚛️ Built with React hooks
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS / Responsive Design
- REST API
- Tailwindcss
